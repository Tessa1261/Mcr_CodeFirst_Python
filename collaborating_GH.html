Collaborating with GitHub

Version Control

Management of changes (called revisions) to any types of information

Advantages of Version Control

  • Can go back to previous version
  • Can work with others without stepping on each others' toes
  • Can add collaborators and contributors
  • Can work from multiple locations

Github

  • Hosts your repositories online
  • Helps you to collaborate with others
  • Web interface to version control....

Interfaces to GitHub

  • Website
  • Command line
  • Desktop apps

Git/GitHub terms

  • Repository [repo]: a project where all of your files are, online or in your laptop
  • Commit: saving a given version of file(s)
  • Fork: make a copy of someone's else repository

Git/GitHub terms

  • Clone: copy a repo to your local computer
  • Pull request: request to add your changes into a forked repo
  • Merge: incorporating new changes from a PR/branch
